## Features

- 표준 계산기
- 전역일 계산기
- 연봉 계산기
- 경력 계산기
- 평 계산기



## Installation

Follow these installation instructions:

### Manually

```bash
$ git clone https://github.com/Connecting-Project/calculator-web.git
$ cd calculator-web
$ npm install 
$ npm start
```

### Docker 

```bash
$ docker run -p 5000:5000 -d jusk2/hawaiian-password-web
```
